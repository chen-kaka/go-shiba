
# go-shiba

基于beego MVC框架的脚手架。

### 内置基础组件：

- 不同环境的不同配置文件

- 配置化CGI

- Mongodb驱动
借鉴[beego-mgo库](https://github.com/goinggo/beego-mgo)

- Statsd上报

- 日志模块



依赖：

### 安装beego
go get github.com/astaxie/beego
### 安装mongodb driver mgo
go get gopkg.in/mgo.v2
### 安装 mgo bson
go get gopkg.in/mgo.v2/bson
